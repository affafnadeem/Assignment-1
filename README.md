
# Saffron House Restaurant App - MVP

## 📱 Overview

A mobile application for Saffron House Restaurant that allows customers to browse menu, book tables, place orders, and track orders. Managers can manage menu items, view orders, and update status.

**Assignment 1 (Frontend Only)**

---

## ✨ Features

### Customer
- 🔐 Register/Login
- 📋 Browse menu with Arabic names
- 🔍 Search and filter by category
- 🔥 View daily specials
- 🛒 Shopping cart with quantity controls
- 📦 Place orders (Dine-in/Takeaway/Delivery)
- 💳 Multiple payment methods
- 📅 Table reservation with QR code
- 📱 Share QR code for reservation/order
- 📊 Track order status
- 📜 Order history

### Manager
- 📊 Dashboard with statistics
- 🍽️ Add/Edit/Delete menu items
- 📋 View and update order status
- 📅 View reservations
- 🪑 View table availability
